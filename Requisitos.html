Casa De Empeños

Requisitos para empeñar

REQUISITOS PARA EMPEÑO DE TERRENOS Y CASAS:

-Nombre completo.

-CURP.

-Correo electronico.

-Domicilio.

-Numero de telefono.

-Valor de la propiedad.

REQUISITOS PARA EMPEÑO DE AUTOS:

-Nombre completo.

-CURP.

-Numero de telefono.

-Año de lanzamiento 1995-2022

-Valor de el automovil.

REQUISITOS PARA EMPEÑO DE ARTICULOS:

-Nombre completo.

-Curp

-Articulo a empeñar.

-Numero de telefono.

-Año de lanzamiento 2015-2022(Solo aparatos Electronicos)

-Valor del articulo.


(El valuador indicara el monto a otorgar segun el estado y caracteristicas del articulo)
